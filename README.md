# DivineDagger

DivineDagger is a lightweight plugin that adds a single custom weapon to your server: the Divine Dagger.
This dagger instantly kills any player it hits and broadcasts a formatted server-wide message on kill.
Designed for PvP servers, events, or any server that wants a unique signature weapon without unnecessary complexity.

## Features
- One fully custom dagger
- One-hit kill mechanic
- Global broadcast on kill
- Simple `/dd give` command
- Lightweight and easy to install

## Command
`/dd give [player]`  
Permission: `core.admin`

## Installation
1. Place the jar file into your `/plugins/` folder
2. Restart the server
3. Use `/dd give` to get the dagger
